Cara kerja singkat (klik untuk lihat)
  1. Ambil input teks, ukuran, margin, dan level error correction.
  2. Rakit URL ke endpoint gambar QR: /v1/create-qr-code?size=....
  3. Setel src pada elemen <img> untuk menampilkan hasil.
  4. Untuk download, gambar dipindahkan ke <canvas> lalu diekspor sebagai PNG.
